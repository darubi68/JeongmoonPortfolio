저는 HTML, CSS, JavaScript를 중심으로 2년 6개월간의 경력을 보유한 웹 퍼블리셔 입니다.
유지 보수가 좋은 코드와 다양한 언어들을 사용하기 위해 지속적으로 학습하며 전문성을 지닌 인재를 목표로 나아가고있습니다.

주로 자사프로그램의 개발과 유지보수를 담당했으며, 이를 통해 실전 경험을 쌓았습니다.
제가 가진 강점 중 하나는 문제 해결 능력과 팀원들과의 원활한 커뮤니케이션입니다. 사용자의 요구사항을 이해하고, 효율적인 솔루션을 찾기 위해 노력합니다. 또한, 팀 내에서 협업하고 의견을 공유함으로써 프로젝트의 품질과 만족도를 높이기 위해 최선을 다합니다.

기획서와 디자인을 꼼꼼히 확인하여 웹 호환성과 표준을 준수한 페이지 구조와 마크업을 기획하며, 깔끔하고 튼튼한 뼈대를 잘 구성합니다.

SCSS를 이용할 수 있고, CSS와 JAVASCRIPT를 사용한 동적인 화면을 구현하는 것에 관심이 높습니다.
요구 결과물에 흡사한 화면을 구현해 낼 수 있으며 반응형 웹 퍼블리싱이 가능합니다.

React와 TypeScript를 활용한 프로젝트 경험이 있고, git과 jira, SVN 등을 이용하여 형상관리를 진행 했습니다. 다양한 언어와 툴을 이용함에 거부감 없이 접근할 수 있습니다.

Photoshop과 Illustrator를 이용하여 적절한 디자인을 구현할 수 있습니다. 직관적이고 편리한 사용자 UI 제공을 최우선으로 여깁니다.

꼼꼼한 성격와 센스있는 감각으로 프로젝트에 능동적으로 참여하고 팀내 분위기 메이커를 담당합니다.
