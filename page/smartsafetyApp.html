스마트간편등록 앱 퍼블리싱

기여도

디자인 100% · 퍼블리싱 100%

기술스택
React
CSS3
Javascript

다양한 기능이 추가 된 근로자간편등록 웹 사이트의 상위 버전 앱입니다. 태블릿 디바이스 사용을 기준으로 제작 되었습니다.

외국인 노동자를 고려한 6개국어 서비스를 제공합니다.
PMS 웹사이트에 관리자가 등록한 동의서, 서약서 등의 문서를 전자화하여 편리하게 관리 할 수 있으며, 등록 한 교육자료를 근로자 개개인이 컨트롤하며 집중도있게 시청할 수 있습니다.

인터넷 강의를 이용하여 REACT 기본적인 사용법을 익힌 뒤 컴포넌트를 이용하여 페이지를 구조화하고 style-components를 이용하여 퍼블리싱 하였습니다.
일러스트를 이용하여 제작한 이미지는 근로자들에게 직관적인 사진 예시를 제공합니다.
클래스와 정규표현식을 이용하여 모든 등록, 수정, 테이블에 정확한 필요값을 입력토록 처리했습니다.
전자화 된 문서가 공통적인 폰트, 색상, 레이아웃으로 출력 되도록 설정하였습니다.
